# Voice Chat (voIP)

A simple **Voice over IP (VoIP)** application built in **Python** using UDP sockets for real-time voice transmission with a graphical user interface.

---

## 🧠 Description

Voice Chat (voIP) is a minimalistic Python voice communication app that sends and receives live audio over a network using `socket`, handles audio input/output via `sounddevice`, and displays a basic interface with `tkinter`.

---

## 🚀 Features

- Real-time audio streaming over UDP sockets
- Simple GUI using `tkinter`
- Uses `sounddevice` for audio capture and playback
- Lightweight and beginner-friendly voice communication

---

## 📋 Requirements

Make sure to have the following installed:

- Python 3.x
- `sounddevice`

Install dependencies with:

```bash
pip install sounddevice
